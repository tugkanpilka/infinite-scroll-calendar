# Contributing

Thanks for contributing to `infinite-scroll-calendar`.

## Local Setup

```bash
yarn install
yarn build
yarn test
yarn storybook
```

## Development Notes

- Keep the public API small and predictable
- Preserve controlled and uncontrolled behavior
- Add or update tests when changing selection, expansion, or rendering logic
- Prefer Storybook updates when a change affects user-facing behavior

## Pull Requests

- Keep changes focused
- Explain the user-facing impact in the PR description
- Include screenshots or Storybook notes for visual changes
- Link related issues when applicable

## Reporting Issues

When opening an issue, include:

- Expected behavior
- Actual behavior
- Reproduction steps
- React version, package version, and browser details when relevant
